**Step 2: Add HeadContent block to Home.razor**

In `Components/Pages/Home.razor`, immediately after the `<PageTitle>@Text.HomeTitle</PageTitle>` line (line 4) and before the `<!-- Hero Section -->` comment, add a single `<HeadContent>` block containing ALL of the following elements:

```razor
<HeadContent>
    <meta name="description" content="Automobilio pirkimo pardavimo sutartis internetu. Užpildykite, pasirašykite ir atsisiųskite — greitai, saugiai, be spausdintuvo." />
    <link rel="canonical" href="https://autodokas.lt/" />
    <meta property="og:title" content="Automobilio pirkimo pardavimo sutartis internetu | AutoDokas" />
    <meta property="og:description" content="Automobilio pirkimo pardavimo sutartis internetu. Užpildykite, pasirašykite ir atsisiųskite — greitai, saugiai, be spausdintuvo." />
    <meta property="og:url" content="https://autodokas.lt/" />
    <meta property="og:type" content="website" />
    <meta property="og:site_name" content="AutoDokas" />
</HeadContent>
```

**What this covers:**
- META-02: `<meta name="description">` -- Lithuanian keywords, ~128 chars (under 155 safety limit), primary keyword in first sentence, communicates benefits (fill in, sign, download), differentiators (fast, safe, no printer)
- META-03: `<link rel="canonical">` -- self-referential with trailing slash (standard for domain-root URLs), absolute URL with https://
- META-04: All 5 OG tags -- og:title mirrors page title, og:description mirrors meta description, og:url matches canonical, og:type=website, og:site_name=AutoDokas. No og:image for v1 (deferred to v2 per user decision).

**Do NOT:**
- Add a `<title>` tag in HeadContent (would create duplicate -- PageTitle handles this)
- Add `<meta name="keywords">` (ignored by Google since 2009)
- Add og:image (deferred to v2, user decision)
